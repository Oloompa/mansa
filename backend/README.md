## Install deps

Install dependencies from root level of monorepo using the following command:

```bash
npm install
```

It will also install some githooks.

## Run

From root level of monorepo

```bash
npm start -w backend
```

From backend workspace

```bash
npm start
```

## Test

From root level of monorepo

```bash
npm test -w backend
```

From backend workspace

```bash
npm test
```
